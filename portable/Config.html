QQ 机器人
已预装插件，只需填 AppID 和 Secret
★ 推荐
获取 QQ 机器人凭证：
  1. 访问 q.qq.com → 登录
  2. 创建一个机器人应用
  3. 在「开发设置」中找到 AppID 和 AppSecret
  4. 把它们填到下面就行
📎
飞书 Feishu
企业首选，内置支持
内置
获取飞书机器人凭证：
  1. 访问 open.feishu.cn → 创建企业自建应用
  2. 在「凭证与基础信息」中获取 App ID 和 App Secret
  3. 在「事件订阅」中配置回调地址
✈️
Telegram
海外推荐，需翻墙
内置
获取 Telegram Bot Token：
  1. 在 Telegram 搜索 @BotFather
  2. 发送 /newbot 创建机器人
  3. 按提示设置名称后，会给你一个 Token
💚
微信 WeChat
社区插件，iPad 协议
社区
微信接入说明：
  1. 微信机器人使用 iPad 协议，需要单独的微信号
  2. 在网页控制台中进入「插件市场」搜索微信插件
  3. 安装后扫码登录即可

⚠️ 微信接入需要在网页控制台中操作

🎮
Discord
游戏社区，内置支持
内置
获取 Discord Bot Token：
  1. 访问 discord.com/developers
  2. 创建 Application → 进入 Bot 页面 → 复制 Token
  3. 在 OAuth2 中生成邀请链接，把 Bot 拉进你的服务器

故障排除

        
遇到问题？试试这些

        

            
Q: 保存后不生效？
点击上面的「应用配置（重载）」，或重启 OpenClaw 启动脚本。

            
Q: 连接不上？
确保启动脚本（Mac-Start.command）正在运行，不要关闭终端窗口。

            
Q: API Key 报错？
检查 Key 是否正确，余额是否充足。去对应平台的控制台查看。

            
Q: QQ 机器人不回复？
1) 确认 AppID/Secret 正确 2) 重启 OpenClaw 3) 在 QQ 开放平台检查机器人状态
